XTT - Xebia Technical Test
## Build 
To build the app:
 ```
 mvnw clean install
 ```
## Run
To execute the app execute the following command after replacing <PATH_TO_INPUT_FILE>(./docs/input.txt for example):
```
java -jar target\xtt-by-hz.jar <PATH_TO_INPUT_FILE>
```
